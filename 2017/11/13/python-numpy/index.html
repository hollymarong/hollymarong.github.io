  • tile(A, dep)
    在各个维度上重复A
    如果A的维度< dep的维度，需要对A增加维度
    tile(A, (a, b, c, d))
    对数组A最深层的维度重复d次，数组A往外一层，重复c次,如A维度不够，增加维度，依次递推。
